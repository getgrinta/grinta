Grinta is a productivity tool that helps you get things done. It's a browser extension that allows you to search the web, take notes, and use natural language to interact with your computer.

## Installation

Download the installer of that latest version of Grinta from our GitHub Releases page.

<LinkButton href="https://github.com/getgrinta/grinta/releases/latest">Download Grinta</LinkButton>

After downloading the installer, double click the downloaded `.dmg` file and drag and drop Grinta to your Applications folder.
