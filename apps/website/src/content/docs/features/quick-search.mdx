Grinta's Quick Search feature allows you to bypass the standard search results and directly search specific websites by typing a shortcut followed by the Tab key.

## How to Use

1.  Type the desired shortcut character (see list below) into the Grinta search bar.
2.  Press the `Tab` key. A badge indicating the active Quick Search provider will appear.
3.  Type your search query.
4.  Press `Enter`. Grinta will open the search results page for that provider in your default browser.

## Available Quick Search Providers

| Shortcut | Provider    | Website                 |
| :------- | :---------- | :---------------------- |
| `G`      | Google      | google.com              |
| `Y`      | YouTube     | youtube.com             |
| `C`      | ChatGPT     | chat.openai.com (via Google) |
| `W`      | Wikipedia   | wikipedia.org           |
| `P`      | Perplexity  | perplexity.ai           |
| `R`      | Reddit      | reddit.com              |
| `F`      | Figma       | figma.com/community     |
| `J`      | Jira        | atlassian.com (via Google) |
| `X`      | X (Twitter) | x.com                   |

To exit Quick Search mode without searching, press the `Escape` key.
